## How to use

1) Running Tests
   At the root of the project:
   - cd .\src\DevFreela.Test\
   - dotnet test

2) Running app with docker
   After start docker, at the root of the project:
   - docker compose up -d

   access: http://localhost:5000/swagger/index.html
